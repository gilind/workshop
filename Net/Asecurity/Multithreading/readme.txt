Написать функцию, которая получает фреймы видео веб камеры с известным http адресом согласно следующим требованиям:
1)     Сеть передачи информации не устойчива: возможны отключения камеры по разным причинам, 
		либо камера может передавать «мусор»
2)     Дан некий объект типа Image obj.
3)     Каждых t сек запускается заданная функция Boolean IsContains(obj, currFrame): если объект содержится в фрейме,
		то в некоторой последовательности(очереди), которую возвращает функция записывается № фрейма и время
4)     Желательно пользоваться минимальным количеством исключений и достигать гарантированную доставку информации.
5)     Данные собираются в течение T    сек. Где t<T.
